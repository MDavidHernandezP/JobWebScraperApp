Hey que pasa mi gente, antes de que hagan cualquier cosa dejen les explico:

Arme el proyecto en un ambiente virtual de python con venv, el que ya trae python por defecto, asi que no armen nada solo apliquen el comando:
(.\sfproject\Scripts\activate) sin los parentesis para levantar el ambiente, despues de eso ya pueden probar a correr la aplicacion,
que esta en el app.py con el ambiente activado. Para desactivar el ambiente solo escriban (deactivate).

Otras cosas:
- En la carpeta (sfproject) esta el ambiente eso no se toca.
- En la carpeta (templates) estan y estaran los futuros templates, osea ahi van los html y solo eso.
- En la carpeta (static) esta otras carpeta llamada (css) donde van los css, es importante esta jerarquia para que se muestren los html con css.

Si para seguir con el resto del projecto ya sea con el scraper o lo que sea, ocupan instalar librerias, instalenlas en el ambiente virtual,
asi una vez que acaben con sus partes y hagan sus commits en sus branches se puedan juntar todas en un super proyecto mamastroso.
